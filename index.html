Commands

Music CommandsNEW

  • $play <song name> - Play a song or add it to queue
  • $stop - Stop playing and clear the queue
  • $skip - Skip the current song
  • $queue - Show the current music queue
  • $pause - Pause the current song
  • $resume - Resume the paused song

Admin Commands

  • $setprefix <new_prefix> - Change the server prefix
  • $add @user <amount> - Add coins to user's balance
  • $banbot @user - Ban user from using bot
  • $unbanbot @user - Unban user from bot
  • $ban @user [time] - Ban user from server
  • $kick @user - Kick user from server
  • $mute @user [time] - Mute a user
  • $unmute @user - Unmute a user
  • $purge <amount> - Delete messages
  • $r @user <role> - Add role to user

Economy Commands

  • $daily - Get daily rewards
  • $balance - Check your balance
  • $gamble <amount> - Gamble your coins
  • $cf <amount> [heads/tails] - Play coinflip
  • $rob @user - Rob another user

Game Commands

  • $hunt - Go hunting
  • $battle @user - Battle another user
  • $petbattle @user - Battle with your team
  • $teamadd - Create your battle team

Inventory Commands

  • $inventory - View your inventory
  • $shop - View the shop
  • $buy <item_number> <quantity> - Buy items

Utility Commands

  • $afk [reason] - Set AFK status
  • $leaderboard - View top players
  • $invite - Get bot invite link

Archanid Bot is your ultimate assistant for managing Discord servers with advanced admin tools, fun games, and daily rewards to keep the community engaged!
